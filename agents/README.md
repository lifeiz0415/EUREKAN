## 파일명 규칙

- 파일명은 `분야_성향_이름.md` 형식을 사용합니다.
- 성향은 `긍정`, `중립`, `부정` 세 값만 사용합니다.
- 예시는 `인공지능_긍정_Elena.md`, `인공지능_중립_Marcus.md`, `인공지능_부정_Vivian.md`입니다.
- 각 파일의 끝에는 `긍정: 긍정적 전망`, `중립: 중립적 전망`, `부정: 부정적 전망` 표시를 둡니다.
- `app.js`의 `deskAgentNames` 작성자 이름 목록은 이 표와 같은 순서로 유지합니다.
