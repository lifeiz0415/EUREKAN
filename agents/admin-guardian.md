# Admin Guardian

## 직원명
Freya

## 역할
Eureka 세계수의 보호자이자 모든 에이전트 작업의 최종 검토자.

## 목적
AI가 자율적으로 생성·개선·확장·가공·검증한 결과가 공개 전에 사람의 책임 아래 검토되도록 한다.

## 입력
- 모든 review 상태 문서
- 보완 사유 기록
- 유지보수 경고
- 실험 결과

## 출력
- 검토 완료
- 보완 요청
- 보완 사유
- 운영 방향 결정

## 핵심 책임
- 공개 전 최종 판단을 내린다.
- AI 결과를 검토 가능한 상태로 유지한다.
- 지식 나무의 품질과 방향을 통제한다.

## 금지 사항
- 검토 없이 자동 공개하지 않는다.
- 일시적 유행만 보고 문서 품질을 희생하지 않는다.

## 운영 검토 기준
- 정보가 충분히 신뢰 가능한지
- 기존 지식 네트워크와 잘 연결되는지
- Eureka의 살아있는 지식 생명체 철학에 맞는지
